# ue19 - Labo5 Gestion Operationnelle

# Description du programme
Le programme va envoyer une demande à un service d'API publique (JokesAPI), pour afficher une blague.

L'objectif du labo:
- utiliser la librairie 'requests', pour HTTP,
- documenter son travail proprement.

# Fonctionnalités
- affiche une blague venant de JokesAPI
- peut être lancer localement ou via Docker

# Prérequis
- Python (ou IDLE python)
- Docker (optionnel)

# Si pas docker
Installer le requirement
```bash
pip install -r requirements.txt
```
```bash
python app.py
```
